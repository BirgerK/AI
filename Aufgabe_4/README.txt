    #######################
    # How to start KiBaNa #
    #######################
	
	
1. 	Im Ordner /Apache-Tomcat-6.0.43/bin in der Konsole die startup.bat 
	aufrufen. Diese Tomcat Instant kann nun im Browser unter 
	http://localhost:8080 erreicht werden.

2. 	Im Ordner /elasticsearch-1.4.1/bin in der Konsole die Datei 
	elasticsearch.bat (unter Windows) oder die Datei elasticsearch (Unix)
	aufrufen. Diese Instanz kann nun unter http://localhost:9200 erreicht 
	werden.
	
3.	Die KiBaNa-Webapp ist nun verfuegbar. Sie kann unter der Addresse
	http://localhost:8080/kibana-3.1.2/ erreicht werden.
